# DicomPack

DICOM sequence packer.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dicompack'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dicompack

### Requirements

* FFmpeg
* ImageMagick
